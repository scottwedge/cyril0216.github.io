学习目标

  1. 能够说出 ls 命令的作用
  2. 能够说出 pwd 命令的作用
  3. 能够说出 touch 命令的作用
  4. 能够说出 mkdir 命令的作用
  5. 能够说出 rm 命令的作用
  6. 能够说出 cd 命令的作用
  7. 能够说出 cp 命令的作用
  8. 能够说出 mv 命令的作用
  9. 能够说出 tree 命令的作用
  10. 能够说出 clear 命令的作用
  11. 能够说出命令自动补全的快捷键

4.1 学习 Linux 终端命令的原因

我们可以通过图形化的界面操作 Linux, 为什么还要学习 Linux 终端命令? 有以下两方面的原因:

  1. Linux 刚出现时没有图形界面, 所有的操作全靠命令完成. 近几年来, 尽管 Linux 发展得非常迅速, 图形界面越来越友好, 但是在真正的开发过程中, Linux 命令行的应用还是占有非常重要的席位.
  1. 在工作中，大量的 服务器维护工作 都是在 远程 通过 SSH 客户端 来完成的，并没有图形界面，所有的维护工作都需要通过命令来完成。如 磁盘操作、文件存取、目录操作、进程管理、文件权限 设定等
远程维护服务器

小贴士:

  1. Linux 发行版本的命令大概有 200 多个, 但常用的命令只有 10 多个.
  2. 不需要死记硬背, 对于常用命令, 用的多了, 自然就记住了.
  3. 不要尝试一次记住所有的命令, 有些命令是非常不常用的, 临时遇到, 临时百度就可以.

4.2 Linux 命令的基本使用

通过图形界面操作完成的任务, 我们也可以通过使用命令来完成. 我们先学习几个常用的命令的基本使用, 对命令的使用有一个初步的认识.

序号 命令 对应英文 作用
01 ls list 查看当前文件夹下的内容
02 pwd print wrok directory 查看当前所在文件夹
03 touch [文件名] touch 如果文件不存在，新建文件
04 mkdir [目录名] make directory 创建目录
05 rm [文件名] remove 删除指定的文件
06 cd [目录名] change directory 切换文件夹
07 cp [文件名] copy 拷贝指定的文件
08 mv [文件名] move 移动指定的文件
09 tree [目录名] tree 以树状方式显示目录结构
10 clear clear 清屏

小技巧:

  1. ctrl + shift + 放大终端窗口的字体显示
  2. ctrl + - 缩小终端窗口的字体显示

4.3 cd 命令切换工作目录

在使用 Unix/Linux 的时候, 经常需要更换工作目录. cd 命令可以帮助用户切换工作目录.

注意: Linux 所有的目录和文件名区分大小写.

4.3.1 cd 命令使用

cd 后面可跟绝对路径, 也可以跟相对路径. 如果省略目录, 默认切换到当前用户的主目录.

命令 含义
cd 切换到当前用户的主目录(/home/用户目录)，用户登陆的时候，默认的目录就是用户的主目录。
cd ~ 切换到当前用户的主目录(/home/用户目录)
cd . 切换到当前目录
cd .. 切换到上级目录
cd - 可进入上次所在的目录

4.3.2 绝对路径和相对路径

  1. 相对路径: 从当前目录算起的路径, 例如当前目录的上一级目录, 当前目录的下一级目录, 都是相对当前目录开始.
相对路径图解
  1. 绝对路径: 以 / 开始的路径.
相对路径图解

4.3.3 命令自动补全

敲出 文件／目录／命令 的前几个字母之后, 按下 tab 键可以进行名称自动补全.

  1. 如果输入的没有歧义, 会自动补全文件、目录、命令的名字.
  2. 如果还存在其他 文件／目录／命令, 再按一下 tab 键, 会提示可能存在的命令.

小提示:

  1. 按 上／下 光标键可以在使用过的命令之间切换.
  2. 如果想要退出选择, 并且不想执行当前选中的命令, 可以按 ctrl + c

4.4 小结

  1. Linux 的 ls 命令用于查看当前文件夹下的内容
  2. Linux 的 pwd 命令用于查看当前所在文件夹
  3. Linux 的 touch 命令用于新建文件
  4. Linux 的 mkdir 命令用于创建目录
  5. Linux 的 rm 命令用于删除指定的文件名
  6. Linux 的 rm 命令用于删除指定的目录
  7. Linux 的 cd 命令用于切换目录
  8. Linux 的 cp 命令用于拷贝指定的目录
  9. Linux 的 mv 命令用于移动指定的文件名
  10. Linux 的 tree 命令用于以树状方式显示目录结构
  11. Linux 的 clear 命令用于清屏
  12. 使用 Tab 键能够自动补全命令
  13. 从当前目录说起的路径称作相对路径
  14. 从根目录算说起路径称作绝对路径
