1. 操作系统

学习目标

  1. 能够说出操作系统的作用
  2. 能够说出 Linux 操作系统的应用领域、优缺点.
  3. 能够说出 Windows 操作系统的应用领域、优缺点.
  4. 能够说出 IOS、Android 操作系统的应用领域.

1.1 操作系统作用

操作系统（Operation System，OS）是管理硬件和控制软件运行的计算机程序, 是直接运行在“裸机”上的系统软件, 任何其他软件都必须在操作系统的支持下才能运行.

  • 主要作用是向下控制硬件向上支持软件的运行，具有承上启下的作用.
  • 大量的软件都直接依赖于操作系统的支持.

1.2 不同应用领域的操作系统

目前操作系统的应用领域主要分为:

  • 桌面操作系统
  • 服务器操作系统
  • 嵌入式操作系统
  • 移动设备操作系统

1.2.1 桌面操作系统

  1. Windows 系列 1.1 用户群体大 1.2 应用软件多
  2. macOS 2.1 适合于开发人员
  3. Linux 3.1 应用软件少

1.2.2 服务器操作系统

机房照片

Linux 在服务器领域的应用是其重要分支.相比 Windows Server, Linux 具有免费、稳定、高效的特点. Linux 在服务器领域的占用率大大高于 Windows Server.

1.2.3 嵌入式操作系统

机顶盒

近些年来 Linux 在嵌入式领域的应用得到了飞速的提高. Linux 运行稳定、对网络的良好支持性、低成本, 且可以根据需要进行软件裁剪, 内核最小可以达到几百 KB 等特点, 使其近些年来在嵌入式领域的应用得到非常大的提高.

手机、机顶盒、数字电视、网络电话、程控交换机、PDA 等都是其应用领域, 并且得到了 Google、三星、摩托罗拉、NEC 等公司的大力推广.

1.2.4 移动设备操作系统

  • iOS
  • Android(基于 Linux)

1.3 小结

  1. 操作系统的作用就是控制硬件和支持软件的运行.
  2. Linux 操作系统应用在服务器领域、嵌入式领域. 其优点如下: 1.1 安全、稳定、免费 1.2 占有率高
  3. Windows 操作系统应用在桌面操作系统领域. 其优点如下: 1.1 用户群体大 1.2 应用软件多
  4. IOS、Android 操作系统主要应用于移动领域.
