3. Navicat

学习目标

  1. 能够使用 Navicat 创建数据库.
  2. 能够使用 Navicat 在数据库中创建数据库表.
  3. 能够使用 Navicat 在数据库中添加数据.

MySQL 数据库本身自带有命令行工具, 也有图形管理工具 MySQL Workbench. 但自带的工具在功能上和易用性上总比不上第三方开发的工具, 比如 Navicat, 使用它你几乎能完成任何针对 MySQL 的管理任务.

Navicat 是一套快速、可靠并价格适宜的数据库管理工具, 适用于三种平台, Microsoft Windows、Mac OS X 及 Linux. 可以用来对本机或远程的 MySQL、SQL Server、SQLite、Oracle 及 PostgreSQL 数据库进行管理及开发. 专为简化数据库的管理及降低系统管理成本而设. 它的设计符合数据库管理员、开发人员及中小企业的需. Navicat 是以直觉化的图形用户界面而建的, 让你可以以安全并且简单的方式对数据库进行操作.

Navicat 图形化客户端

下面我们使用 Navicat 从三方面来介绍该数据库客户端工具的使用:

  1. 数据库操作.
  2. 数据表的操作.
  3. 数据库表中数据的操作.

  • 选择一张表后，工具栏的第二行“打开表”、“设计表”、“删除表”都变的可用
创建数据表 第1步

  • 设计表和创建表的窗口一样，可以增加、修改、删除字段，或编辑字段的类型、约束
  • 删除表会将表物理删除

  • 如果需要继续添加数据，点击询问的加号，会出现一个新的空白行，填写数据即可

3.5 总结

在工作中为了提高开发效率往往需要采用一些数据库工具, 但是为了夯实同学们的基础, 能够在面试和工作中在处理数据的时候得心应手, 我们在很多情况下还需要熟悉命令行客户端的操作-本质上就是SQL语句的编写.
