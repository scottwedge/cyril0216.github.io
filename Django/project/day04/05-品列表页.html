商品列表页

当用户点击了商品分类中的'手机' 这个三级分类的时候, 我们就会进入商品列表页面

就会进入下面的页面:

网址中的 cat参数 是商品第三级菜单的 id值，即, 当前列表页展示的内容既是 cat参数 筛选的内容

获取商品列表页数据接口

业务需求

对于商品列表页, 我们需要完成以下需求:

  • 提供分页支持
  • 可以按照时间( 默认的 ) 排序
  • 可以按照价格排序
  • 可以按照销量( 人气 ) 排序

接口设计

请求方式：

# 我们会发现请求的参数中不但传入了想要获取的 id值
# 也传入了请求的页码, 当前页容量 和排序关键字
GET   /categories/(?P<category_id>\d+)/skus?page=xxx&page_size=xxx&ordering=xxx

请求参数： 路径参数 + 查询字符串参数

参数 类型 是否必须 说明
categroy_id int 是 类别id（第三级类别）
page int 是 页数
page_size int 否 每页数量
ordering str 是 排序关键字（'create_time', 'price', 'sales'）

后端实现

第一步: 我们先来解决排序问题.

说到排序, 我们在基础部分已经学习过排序了

所以我们可以定义一个排序类, 来帮助我们实现排序功能:

在 meiduo_mall / utils 中创建 pagination.py 文件，在其中创建分页配置类

from rest_framework.pagination import PageNumberPagination

class StandardResultPagination(PageNumberPagination):
    # 默认的每页加载个数,如果没有提供,按照这个来进行
    page_size = 2
    # 每页加载个数使用的关键字
    page_size_query_param = 'page_size'
    # 每页最大加载个数
    max_page_size = 20

在配置文件中设置 REST framework 分页使用的分页类

REST_FRAMEWORK = {
    ...
    # 使用我们添加的分页类来对页面进行分页
    'DEFAULT_PAGINATION_CLASS':'meiduo_mall.utils.pagination.StandardResultPagination',
}
第二步: 我们来解决序列化器问题:

序列化器采用之前实现的 SKUSerializer, 因为该接口还是需要从数据库中取出商品, 并且把商品序列化为 json 返回给前端. 所以使用商品的序列化器即可.

第三步: 我们来解决视图问题:

既然获取商品的接口, 我们就把该视图添加到 goods 子应用的 views.py 文件中.

在 goods / views.py 中实现视图

from rest_framework.filters import OrderingFilter

# 定义获取列表页的视图:
class SKUListView(ListAPIView):
    """
    sku列表数据
    """
    serializer_class = SKUSerializer
    # 使当前的类视图支持一下三种排序方式: create_time,price,sales
    filter_backends = (OrderingFilter,)
    ordering_fields = ('create_time', 'price', 'sales')

    def get_queryset(self):
        # 我们从前端传过来的所有数据, 最终都传给了 self.kwargs 中.
        # 所以我们可以通过 self.kwargs 来获取 category_id
        category_id = self.kwargs['category_id']
        # 然后把SKU表中三级菜单id相同的,以及上架的都获取出来上传
        return SKU.objects.filter(category_id=category_id, is_launched=True)

说明：

REST framework 提供了对于排序的支持，使用 REST framework 提供的 OrderingFilter 过滤器即可

OrderingFilter 过滤器要使用 ordering_fields 属性来指明可以进行排序的字段有哪些

第四步: 我们来解决路由问题:
# 总路由: 
    url(r'^', include('goods.urls')),


# 子路由: 
    url(r'^categories/(?P<category_id>\d+)/skus/$', views.SKUListView.as_view())

var vm = new Vue({
    el: '#app',
    delimiters: ['[[', ']]'], // 修改vue模板符号，防止与django冲突
    data: {
        host: host,
        username: sessionStorage.username || localStorage.username,
        user_id: sessionStorage.user_id || localStorage.user_id,
        token: sessionStorage.token || localStorage.token,
        cat: '', // 当前商品类别
        page: 1, // 当前页数
        page_size: 5, // 每页数量
        ordering: '-create_time', // 排序
        count: 0,  // 总数量
        skus: [], // 数据
        cat1: {url: '', category:{name:'', id:''}},  // 一级类别
        cat2: {name:''},  // 二级类别
        cat3: {name:''},  // 三级类别,
        cart_total_count: 0, // 购物车总数量
        cart: [], // 购物车数据
        hots: [], // 热销商品
    },
    computed: {
        total_page: function(){  // 总页数
            return Math.ceil(this.count/this.page_size);
        },
        next: function(){  // 下一页
            if (this.page >= this.total_page) {
                return 0;
            } else {
                return this.page + 1;
            }
        },
        previous: function(){  // 上一页
            if (this.page <= 0 ) {
                return 0;
            } else {
                return this.page - 1;
            }
        },
        page_nums: function(){  // 页码
            // 分页页数显示计算
            // 1.如果总页数<=5
            // 2.如果当前页是前3页
            // 3.如果当前页是后3页,
            // 4.既不是前3页，也不是后3页
            var nums = [];
            if (this.total_page <= 5) {
                for (var i=1; i<=this.total_page; i++){
                    nums.push(i);
                }
            } else if (this.page <= 3) {
                nums = [1, 2, 3, 4, 5];
            } else if (this.total_page - this.page <= 2) {
                for (var i=this.total_page; i>this.total_page-5; i--) {
                    nums.push(i);
                }
            } else {
                for (var i=this.page-2; i<this.page+3; i++){
                    nums.push(i);
                }
            }
            return nums;
        }
    },
    mounted: function(){
        this.cat = this.get_query_string('cat');
        this.get_skus();
        axios.get(this.host+'/categories/'+this.cat+'/', {
                responseType:'json'
            })
            .then(response => {
                this.cat1 = response.data.cat1;
                this.cat2 = response.data.cat2;
                this.cat3 = response.data.cat3;
            })
            .catch(error => {
                console.log(error.response.data)
            });
        this.get_cart();
        this.get_hot_goods();
    },
    methods: {
        logout(){
            sessionStorage.clear();
            localStorage.clear();
            location.href = '/login.html';
        },
        // 获取url路径参数
        get_query_string: function(name){
            var reg = new RegExp('(^|&)' + name + '=([^&]*)(&|$)', 'i');
            var r = window.location.search.substr(1).match(reg);
            if (r != null) {
                return decodeURI(r[2]);
            }
            return null;
        },
        // 请求商品数据
        get_skus: function(){
            axios.get(this.host+'/categories/'+this.cat+'/skus/', {
                    params: {
                        page: this.page,
                        page_size: this.page_size,
                        ordering: this.ordering
                    },
                    responseType: 'json'
                })
                .then(response => {
                    this.count = response.data.count;
                    this.skus = response.data.results;
                    for(var i=0; i<this.skus.length; i++){
                        this.skus[i].url = '/goods/' + this.skus[i].id + ".html";
                    }
                })
                .catch(error => {
                    console.log(error.response.data);
                })
        },
        // 点击页数
        on_page: function(num){
            if (num != this.page){
                this.page = num;
                this.get_skus();
            }
        },
        // 点击排序
        on_sort: function(ordering){
            if (ordering != this.ordering) {
                this.page = 1;
                this.ordering = ordering;
                this.get_skus();
            }
        },
        // 获取购物车数据
        get_cart: function(){
            axios.get(this.host+'/cart/', {
                    headers: {
                        'Authorization': 'JWT ' + this.token
                    },
                    responseType: 'json',
                    withCredentials: true
                })
                .then(response => {
                    this.cart = response.data;
                    this.cart_total_count = 0;
                    for(var i=0;i<this.cart.length;i++){
                        if (this.cart[i].name.length>25){
                            this.cart[i].name = this.cart[i].name.substring(0, 25) + '...';
                        }
                        this.cart_total_count += this.cart[i].count;

                    }
                })
                .catch(error => {
                    console.log(error.response.data);
                })
        },
        // 获取热销商品数据
        get_hot_goods: function(){

        }
    }
