保存支付结果

用户支付成功后，支付宝会将用户重定向到http://www.meiduo.site:8080/pay_success.html，并携带支付结果数据

前端页面将此数据发送给后端，后端检验并保存支付结果

1. 后端接口设计

请求方式： PUT /payment/status/?支付宝参数

请求参数： 查询字符串参数， 见上面表格

返回数据： JSON

返回值 类型 是否必须 说明
trade_id str 否 支付宝流水号

后端实现

在 payment / views.py 中创建视图

# put请求, 格式: /payment/status/?支付宝参数
class PaymentStatusView(APIView):
    """支付结果"""

    # 检验是否是登录用户: 
    permission_classes = [IsAuthenticated]

    def put(self, request):
        # 1.获取请求路径上的查询字符串参数: 
        # 并把该参数转为字典: .dict()
        data = request.query_params.dict()
        # 从获取的data字典中, 去掉 sign, 得到sign
        signature = data.pop("sign")

           # 2. 初始化: 
        alipay = AliPay(
            appid=settings.ALIPAY_APPID,
            app_notify_url=None,  
            app_private_key_path=os.path.join(os.path.dirname(os.path.abspath(__file__)), "keys/app_private_key.pem"),
             # 支付宝的公钥:
            alipay_public_key_path=os.path.join(os.path.dirname(os.path.abspath(__file__)),
                                                "keys/alipay_public_key.pem"), 
            # RSA 或者 RSA2 加密方法: 
            sign_type="RSA2",  
            # 是否使用沙箱, 默认为True
            debug=settings.ALIPAY_DEBUG  
        )

        # 3. 进行验证: 
        success = alipay.verify(data, signature)

        # 4. 判断签名是否正确: 
        if not success:
            return Response({'message': '非法请求'}, status=status.HTTP_403_FORBIDDEN)

        # 获取订单编号
        order_id = data.get('out_trade_no')

        # 5. 校验订单id(order_id)
        try:
            order = OrderInfo.objects.get(
                order_id=order_id,
                user=request.user,
                status=OrderInfo.ORDER_STATUS_ENUM['UNPAID']
            )
        except OrderInfo.DoesNotExist:
            return Response({'message': '订单信息有误'}, status=status.HTTP_400_BAD_REQUEST)


        # 支付宝流水号(交易号)
        trade_id = data.get('trade_no')

        # 6. 添加支付记录
        Payment.objects.create(
            order=order,
            trade_id=trade_id
        )

        # 7. 更新订单的支付状态
        order.status = OrderInfo.ORDER_STATUS_ENUM['UNSEND']
        order.save()

        return Response({'trade_id': trade_id})

路由:

# 订单的路由信息: 

url(r'^payment/status/$', views.PaymentStatusView.as_view()),
