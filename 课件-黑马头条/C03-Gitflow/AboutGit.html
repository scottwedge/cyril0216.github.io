冲突解决

  • 出现冲突的原因:
    • 多人修改了同一个文件同一行

解决示例

~ 提交合并请求时, 会提示出现冲突
~ 先取消合并请求
git checkout develop
git pull
git checkout f_order
git merge develop
git add 
git commit 
git push
~ 重新发起合并请求
