3. 开发工具

CentOS7.2虚拟机

  • 作为项目的开发环境

  • 黑窗口 无GNOME 模拟企业中的开发服务器环境

  • 网络配置

    • 开机前先将虚拟机调成NAT模式
    • 虚拟机占用内存4G，如果主机内存不足，可启动前将内存占用调为2G
  • 远程登录
    • ssh python@自己的ip地址
    • 没有内置ssh工具的系统 使用 Termius、xshell等ssh客户端工具
  • 用户名&密码
    • 系统
      • root -> chuanzhi
      • python -> chuanzhi
    • MySQL
      • root -> mysql
  • 端口
    • MySQL (mariadb)
      • master -> 3306
      • slave -> 8306 (mysql -uroot -p -h 127.0.0.1 --port=8306)
    • Redis
      • cluster -> 7000 7001 7002 7003 7004 7005
      • master & slave -> 6380 6381
      • sentinel -> 26380 26381 26382
    • Elasticsearch 5
      • 9200
  • Python 虚拟环境
    • workon toutiao
  • 命令
    • 关机 sudo shutdown now
    • 重启 reboot

Gitlab

  • 私有代码托管平台, 开源免费

  • 请到http://git.meiduo.site注册帐号

  • 为方便将大家拉入到项目团队中，注册帐号的要求

  • 使用Windows的同学需要在windows中安装git

  • 注意：此gitlab不要放私人项目，会定期清理数据。

YApi 接口管理

  • 在线接口管理平台, 开源免费

  • http://mock.meiduo.site
