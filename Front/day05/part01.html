Introduction

数组定义

数组就是一组数据的集合

javascript 中, 数组就相当于我们 python 中的列表, 里面的元素可以是不同类型的

定义数组的方法

数组定义形式: 
第一种: 
var 数组名 = new Array(元素一, 元素二, 元素三...) 


第二种: 
var 数组名 = [元素一, 元素二, 元素三...]

例如: 

// 对象的实例创建
var aList = new Array(1,2,3);

// 直接量创建
var aList2 = [1,2,3,'asd'];

数组常见的操作方法

1、获取数组的长度：

一般我们获取数组长度使用: length

这里需要大家, 我们使用的不是方法,而是一个属性, 所以 length 后面不能加小括号

使用格式: 

数组名.length

例如: 

var aList = [1,2,3,4];

// 打印输出数组 aList 的长度
console.log(aList.length)

结果: 

4

练习: var array01 = ['人生自古谁无死', '哪个拉屎不用纸']

2、下标索引的使用：

和python一样, 我们也可以利用下标索引的思想来操作数组中某一个元素: aList[0]

下标使用时从左侧开始.

使用格式: 

array[下标值]

例如: 

var aList = [1,2,3,4];

// 查看0号下标的值
console.log(aList[0]);

结果: 

1

练习: var array02 = ['一般般的我', '一般般的亮', '一般般的你', '我看不上']

3、join(分隔符)

作用: 将数组变为字符串

将数组成员通过一个分隔符合并成字符串

使用方式: 


数组.join(分隔符)

例如: 

var aList = [1, 2, 3, 4]

var newList = aList.join('-')

console.log(newList)
console.log(typeof(newList))

结果: 

1-2-3-4
string

练习: var array03 = ['我自横刀向天笑', '笑完我就去睡觉']

4、push( ) 和 pop( )

push( ) 给数组增加元素

pop( ) 给数组去除元素

都是从数组的最后进行操作: 从最后增加, 从最后去除.

使用方式: 


数组.push(新元素)

数组.pop()       // 从最后去除元素

例如: 

var aList = [1,2,3,4];

aList.push(5);
console.log(aList)

aList.pop()
console.log(aList)

结果:

[1, 2, 3, 4, 5]

[1, 2, 3, 4]

练习: var array04 = ['美好的你', '美好的他', '美好的生活靠大家']

5、reverse( )

将数组反转

注意: 是反转, 不是正向排序, 也不是倒向排序.

使用方式: 

数组.reverse()

例如: 

var aList = [3,2,1,4];

aList.reverse();
console.log(aList)

结果: 

[4, 1, 2, 3]

练习: var array05 = ['颠覆的日子', '让我没有了黑夜白天']

6、indexOf( )

返回数组中元素第一次出现的索引值:

  • 如果有, 则正常返回索引值
  • 如果检索的内容不存在于数组中, 则返回 -1
  • 程序不会崩溃报错.只会返回-1
使用方式: 

数组.indexOf(检测的元素)

例如: 

var aList = [1,2,3,4,1,3,4];

console.log( aList.indexOf(4) )
console.log( aList.indexOf(5) )

结果: 

3
-1

练习: var array06 = ['别人手牵手', '我牵我的狗', '走一走', '游一游', '看谁不爽咬两口']

7、splice( )

在数组中增加或删除成员

如果没有第三个参数,则是删除方法

如果有第三个参数, 则是增加方法

使用格式: 

数组名.splice(index,howmany,item1,.....,itemX)

index: 必需。整数，规定添加/删除项目的位置，使用负数可从数组结尾处规定位置。

howmany: 必需。要删除的项目数量。如果设置为 0，则不会删除项目。

item1, ..., itemX: 可选。向数组添加的新项目。

例如:

var aList = [1,2,3,4];

// 从第2个元素开始，删除1个元素，然后在此位置增加'7,8,9'三个元素
aList.splice(2,1,7,8,9); 
alert(aList); //弹出 1,2,7,8,9,4

练习: var array07 = ['别人手牵手', '我牵我的狗', '走一走', '游一游', '看谁不爽咬两口']

8. 多维数组

多维数组指的是数组的成员也是数组的数组。

var aList = [[1,2,3],['a','b','c']];

alert(aList[0][1]); //弹出2;
