Introduction

函数

函数就是重复执行的代码片段。

函数定义与调用

<script>

    // 函数定义
    function fnAlert(){
     alert('hello!');
    }

    // 函数调用
    fnAlert();
</script>

预解析

函数和变量都有预解析功能

函数的预解析和原来的使用一样,没有区别

变量的预解析只是把变量的声明提前了,变量的赋值还是在原来的位置.

JavaScript 解析过程分为两个阶段，先是编译阶段，然后执行阶段，在编译阶段会将function定义的函数提前，并且将var定义的变量声明提前，将它赋值为 undefined。

函数预解析功能    使用效果: 

// 函数的调用
func()


function func () {
    // 这是函数的定义
}

变量预解析功能   使用效果: 

alert(num)     // undefined  不报错, 显示没有赋值,而不是没有定义

var num = 10

例如: 

<script>    
    // 函数预解析: 

    fnAlert();       // 弹出 hello！

    function fnAlert(){
        alert('hello!');
    }


    // 变量预解析:

    alert(iNum);  // 弹出 undefined

    var iNum = 123;

</script>

函数参数

Javascript 的函数中可以传递参数

<script>
    // 定义函数的时候, 可以设置形参
    function fnAlert(a,b){
        alert(a);
    }

    // 调用函数的时候传递实参
    fnAlert(12345,123);

</script>

函数的返回值

JavaScript 中函数使用 return 返回内容

函数中 'return' 关键字的作用：

  • 返回函数中的值或者对象
  • 结束函数的运行
  • 特殊的: return false (在阻止事件冒泡时使用)
<script>

    // 函数的定义
    function addFunc(num1, num2){
        var result = num1 + num2;
        return result;
        alert('here!');
    }

    // 函数的调用
    var result = addFunc(3,4);
    // 弹出7
    alert(result);  

</script>
