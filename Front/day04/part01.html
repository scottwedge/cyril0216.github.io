Introduction

JavaScript 介绍

JavaScript 是一种直译式脚本语言。

它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端.

最早是在HTML（网页)上使用，用来给HTML网页增加动态功能。

动态功能包括: 部分动画效果、页面与用户的交互

前端各部分的作用:

HTML： 决定页面结构是否存在

CSS： 决定页面显示效果

JavaScript： 决定页面中人机交互功能

引入 JavaScript 的三种方式:

1、行间添加

<!--可以把 js 的内容添加到标签内部-->

<div onclick="alert('hello world')">

2、嵌套方式添加

<head>
    <!--我们在head部分添加一个script标签, 内部添加js内容-->
    <script>        
            alert('ok！');
    </script>
</head>

3、外部引入方式

<!-- 我们可以在外部创建一个独立的js文件.  使用script标签导入 -->

<script type="text/javascript" src="js/index.js"></script>

总结:

  • JS 有三种导入方式: 分别是在标签内部写入. 在script标签内部导入. 单独创建一个js文件.
  • 我们后面会多次使用 JS 的导入, 所以这部分需要掌握.
